
## 🚀 Quick Install (Auto-Installer)

**Cara termudah - 1 command saja!**

```bash
# Clone & Install otomatis
git clone https://github.com/Septian696/supa-cli.git
cd supa-cli
./install.sh
```

**Atau download ZIP:**

```bash
# Download ZIP
curl -L https://github.com/Septian696/supa-cli/archive/refs/heads/main.zip -o supa-cli.zip
unzip supa-cli.zip
cd supa-cli-main
./install.sh
```

Setelah install, jalankan:
```bash
./supa
# OR
node supa.mjs
```
